# FlowBridge blog starter

This version removes the Insights section completely and changes Resources to Blog.

## What it uses

- Static HTML pages for Home, About, and Services
- A blog index at `/blog/`
- Pages CMS for blog posts only
- Jekyll custom collection stored in `_blog`

## Add a blog post

1. Open Pages CMS
2. Choose **Blog**
3. Add a new entry
4. Fill in the title, date, category, cover image, excerpt, and content
5. Publish

The post will appear automatically on the Blog page.
